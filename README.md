# Trie进阶（应用于API路由）

字符种类：26个大写字母, 26个小写字母, "/", "*", '{', "}"
共计：26+26+4 = 56

对应ASCII码（十进制）：
"/" : 47
"*": 42
"{": 123
"}": 125
"A"-"Z": 65-90
"a"-"z": 97-122

## version 1.0

Trie树构造方案：用ASCII码区分不同的字符，同时不考虑空间开销，每一个Trie树节点都有128个（有可能）的子节点
